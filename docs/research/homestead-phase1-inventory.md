## Key Findings

### 1. The "cozy feel" is built from 5 stacked sensory layers

| Layer | Primary mods |
|-------|-------------|
| **Sound** | AmbientSounds + Sound Physics Remastered (+ Presence Footsteps on Fabric) |
| **Light** | LambDynamicLights + Iris/Euphoria shaders + Decorative Lamps |
| **Visual clutter** | Supplementaries + Handcrafted + Chipped + Macaw's suite |
| **Food culture** | Farmer's Delight family + Let's Do suite (cooking as roleplay loop) |
| **World richness** | YUNG's suite + Tectonic + Nature's Spirit (rewarding to explore) |

### 2. NeoForge gaps — highest priority to resolve

| Gap | Fabric mod | NeoForge alternative to investigate |
|-----|-----------|--------------------------------------|
| Core cooking system | Farmer's Delight Refabricated | Farmer's Delight (original, Forge/NeoForge) |
| Footstep sounds | Presence Footsteps | Sound Physics covers reverb; needs investigation |
| Ambient particles | Visuality | Subtle Effects partially covers this |
| Custom sky | FabricSkyBoxes | No direct equivalent yet |
| Accessory system | Trinkets | Curios API (NeoForge native) |
| Cave overhaul | Spelunkery | No confirmed NeoForge port |
| Dense storage | Storage Drawers | Need to check NeoForge availability |
| Connected textures | Continuity | ✅ has NeoForge port |

### 3. The "Delight" ecosystem needs a NeoForge audit
Most addons (Ocean's, End's, Nature's, Chef's, Crate) have NeoForge ports. The core `Farmer's Delight` NeoForge version has its own separate addon ecosystem that may cover the same ground differently.

### 4. Create is a minor ingredient in Homestead, not the main course
Homestead uses Create for flavor (Steam 'n' Rails, Deco). CozyCreate would elevate it to the primary progression loop — this is a major design departure, not just a mod addition.
