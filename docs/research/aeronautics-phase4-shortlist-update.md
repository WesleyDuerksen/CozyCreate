## Swaps & Removals First

These changes modify the existing Phase 4 shortlist before additions are listed.

| Action | Mod Out | Mod In | Reason |
|--------|---------|--------|--------|
| 🔄 Swap | Create: Track Map (`create-track-map`) | Create Railways Navigator (`create-railways-navigator`) | Navigator provides richer multiplayer HUD (schedules, arrivals, routes) and covers the map use case. Track Map is superseded. |
