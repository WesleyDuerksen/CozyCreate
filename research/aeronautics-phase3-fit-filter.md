## QoL & Multiplayer

| Mod | Verdict | Rationale |
|-----|---------|-----------|
| AppleSkin | ✅ Add | Hunger and saturation values on hover. Essential in a food-rich pack where players are cooking and eating diverse items. |
| Backpacked | ❌ Skip | We have Sophisticated Backpacks. Player-confirmed cut. |
| Bad Wither No Cookie | ✅ Add | Suppresses boss spawn sounds server-wide. In multiplayer, one player triggering a boss audio event should not interrupt everyone else's ambient experience. |
| Biome Music | ❌ Skip | Investigated: Biome Music is a library — it does nothing without a custom resource pack. AoCA ships their own branded resource pack to drive it. Without that pack, the mod is inert. AmbientSounds covers our environmental audio layer. |
| Caelus API | ✅ Add | Required dependency for Elytra Slot, which is already in our shortlist. |
| Chat Heads | ✅ Add | Player head icon next to chat messages. In a multiplayer pack, recognizing who's talking at a glance is a genuine quality-of-life improvement. |
| Cut Through | ❌ Skip | Mining through multiple aligned blocks in one swing. This is a techpack feature — it optimizes resource extraction as a mechanical loop. CozyCreate's mining is meant to motivate exploration and feed automation, not to be optimized as the activity itself. |
| Despawn Tweaks | ✅ Add | Configurable item despawn timers. Essential for a server where players are running automation that drops large item quantities — a sensible despawn window prevents lag without losing items unfairly. |
| Easy Anvils | ✅ Add | Removes the anvil level cap and per-use cost scaling. Enchanting is deeply integrated with Create: Enchantment Industry in our pack. Players should be able to engage with that system without hitting arbitrary anvil limits. |
| Easy Magic | ✅ Add | Re-roll enchanting table options. Pairs naturally with Enchantment Industry — the automated enchanting system needs a way to target specific enchantments, and re-rolling is the tactile version of that. |
| Enchantment Descriptions | ✅ Add | Text descriptions for enchantments in tooltips. In a pack with this much automation around enchanting, players should understand what they're producing. |
| FTB Filter System | ✅ Add | Advanced item filter UI. We already have FTB Quests — FTB Filter System is the routing/sorting companion that makes automation sorting practical in a pack this large. |
| FTB XMod Compat | ✅ Add | Compatibility bridges between FTB mods and the rest of the pack. Required for FTB systems to interact with Create and other mods cleanly. |
| Just Enough Breeding | ✅ Add | JEI add-on for breeding info. In a pack with realistic bees, ducks, and diverse animals, breeding information in JEI is practical QoL. |
| Just Enough Professions | ✅ Add | JEI villager trade info. We have Trading Post and Chef's Delight adding professions — players need to know what trades exist. |
| Just Enough Resources | ✅ Add | Drop rates and world gen info in JEI. Essential reference for a pack this deep — players need to know where to find specific materials and resources. |
| NetherPortalFix | ✅ Add | Fixes nether portal linking in multiplayer. Without this, multiple players using separate portals get sent to the same nether destination. Standard multiplayer fix. |
| Neruina | ✅ Add | Prevents a single broken ticking entity from crashing the server. This is critical for a multiplayer-first pack — automation mods can spawn edge cases that crash vanilla servers. |
| Pick Up Notifier | ✅ Add | Toast on item pickup. Small but satisfying — collecting materials from an automated farm feels tangibly rewarding with pickup feedback. |
| Skin Layers 3D | ✅ Add | 3D outer skin layer. Cosmetic, but multiplayer presence is richer when players look more distinct. Low performance cost. |
| Trading Post | ✅ Add | Access multiple villagers' trades from one block. A cozy market or bazaar room becomes a realistic design space — one block invites you to build a proper trading hall around it. |
| Visual Workbench | ✅ Add | Items visible on crafting table surface. Tactile and satisfying — you see what you're making. Cozy, low-tech delight. |
| What Are They Up To | ✅ Add | See other players' held items above their heads. Multiplayer awareness — knowing at a glance that your friend is flying an airship or running a farm adds to the living-world feel. |

## Performance

| Mod | Verdict | Rationale |
|-----|---------|-----------|
| AllTheLeaks | ✅ Add | Memory leak collection fix. A pack this size will have leaks. This is cheap insurance. |
| Create Better FPS | ✅ Add | Create-specific rendering optimizations. We're running a Create-heavy pack — this is targeted and low-risk. |
| GPU Memory Leak Fix | ✅ Add | GPU-side memory leak patching. Same logic as AllTheLeaks. |
| Leaky | ✅ Add | Item entity rendering lag fix. Automation packs drop many items. This is directly relevant. |
| Smooth Chunk Save | ✅ Add | Async chunk saving. On a multiplayer server with heavy automation and contraption movement, synchronous chunk saves cause noticeable lag spikes. This is a meaningful server-side improvement. |
| Sodium Dynamic Lights | ✅ Add (swap) | Investigated: SDL is an unofficial port of LambDynamicLights purpose-built for the Sodium/Embeddium pipeline. Covers the same use cases (torches, fireflies, embers) with tighter renderer integration. LambDynamicLights is replaced by SDL in our stack. Flag for Phase 4 shortlist update. |
| Sinytra Connector | ❌ Skip | Runs Fabric mods on NeoForge. We've committed to the NeoForge-native path. Adding a Fabric compatibility shim introduces a layer of fragility (update cycles between Connector, Fabric API, and individual mods don't always align). The mods we've chosen that were Fabric-only have been substituted or accepted as gaps. |
| Forgified Fabric API | ❌ Skip | Required only by Sinytra Connector. Skipped with it. |
