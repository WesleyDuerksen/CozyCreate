# CozyCreate Mod Shortlist — Phase 4

> Date: 2026-05-08  
> Based on: Homestead analysis (Phases 1–3) + new additions for CozyCreate's steam/diesel-punk vision  
> Target: Minecraft 1.21.1, NeoForge  
> Legend: ✅ confirmed NeoForge | ⭐ new addition (not in Homestead) | 🔄 swap from Homestead equivalent

All mods listed here have confirmed NeoForge support unless marked otherwise.

---

## Create Ecosystem (the spine of CozyCreate)

These mods define what makes CozyCreate different from Homestead. Every mod in this group either adds engineering content or helps Create builds look cozy rather than purely industrial.

| Mod | Slug | Role |
|-----|------|------|
| Create | create | Core automation engine |
| Create: Aeronautics | create-aeronautics | ⭐ Airships, planes, vehicles — the exploration pillar |
| Create: Diesel Generators | create-diesel-generators | ⭐ Diesel power — fits steam/diesel-punk tone perfectly |
| Create: Steam 'n' Rails | create-steam-n-rails | Railways and steam systems |
| Create: Slice & Dice | slice-and-dice | Automates Farmer's Delight via Create machines |
| Create: Bells & Whistles | bellsandwhistles | ⭐ 36 decorative blocks for train stations and engineering spaces — bridges industrial and cozy |
| Create Deco | create-deco | Industrial decoration that fits the Create aesthetic |
| Create: Copycats+ | copycats | Dress any Create block in any texture — essential for cozy builds |
| Create: Enchantment Industry | create-enchantment-industry | ⭐ Automates enchanting through Create — keeps progression in engineering |
| Create: Dreams & Desires | create-dreams-and-desires | ⭐ QoL and decorative pieces that fit Create's visual language |
| Create: Garnished | create-garnished | ⭐ Nut trees and food items that feed into Create automation |
| Create: Track Map | create-track-map | ⭐ Live web map of your rail network — multiplayer-first feature |
| Create: Power Grid | power-grid | ⭐ Electrical distribution network — completes the power ecosystem alongside Diesel Generators |
